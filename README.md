**Ubegood** adalah landing page modern untuk produk **Ubi Oven Premium** dengan berbagai topping lezat.

Website ini dirancang dengan fokus pada:

* 🎨 Visual yang menarik
* 📱 Responsive Design
* ⚡ Fast Loading
* ✨ Smooth Animation
* 🛒 User-Friendly Ordering Experience

# ⚡ Quick Start

Clone repository:

```bash
git clone https://github.com/username/ubegood.git
```

Masuk ke folder project:

```bash
cd ubegood
```

Jalankan server PHP:

```bash
php -S localhost:8000
```

Buka browser:

```text
http://localhost:8000
```

# 🎨 Customization

<details>
<summary><b>Click to Expand</b></summary>

### Mengubah Produk

Edit data menu pada:

```php
index.php
```

### Mengubah Warna

Cari class Tailwind:

```html
bg-orange-600
```

dan ganti sesuai kebutuhan.

### Mengubah Animasi

Cari:

```javascript
AOS.init()
```

Kemudian sesuaikan:

* duration
* easing
* delay
* offset
