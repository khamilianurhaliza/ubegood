## 🖼️ Website Screenshots

The following screenshots showcase the user interface and overall experience of the **Ubegood Website**, a modern landing page developed to support the digital promotion of premium baked sweet potato products.

This project focuses on delivering an engaging visual experience, clear product presentation, intuitive navigation, and responsive performance across various devices. The website combines modern UI design principles with smooth animations to create a professional and attractive digital presence for local food businesses.

### 🎯 Project Highlights

* 🍠 Premium product branding and presentation
* 🎨 Modern and visually appealing user interface
* 📱 Fully responsive design across devices
* ⚡ Fast-loading and lightweight architecture
* ✨ Interactive animations powered by AOS
* 🛒 Easy ordering flow through WhatsApp integration
* 💡 User-centered design approach
* 🚀 Optimized landing page structure for marketing purposes
